podemos criar textos com marcações

Negrito e Italico

podemos criar texto em negrito com a tag b mas ela não é sematinca, por isso hoje em dia se usa a tag strong

também podemos criar texto em italico com a tag i mas não é semantica, por isso utilizamos hoje em dia a tag em

textos marcados e misturas de css

é possível marcar textos com a tag mark

mas também podemos editar a cor da marcação adicionando sobre a tag mark a tag style e usar a configuração bakground-color para mudar a cor da marcação por meio da css

textos grandes e pequenos

é possível aumentar e diminuir um texto por meio das tags big e small

porém, a tag big não possui sentido de significado apenas de visual por isso ela aparece como vermelha dentro do visual estudio code

textos deletados e inseridos

existem também textos que são deletados significando que eles podem ser lidos mas devem ser desconsiderados pelo leitor (antigamente existia a tag strike mas hoje ela foi descartada)

também existem as tag inseridas que são indicadas como importantes e que devem ser lidas

textos sobrescritos e subscritos

os textos sobrescritos são utilizados com intuito de sobrepor um número à uma letra como em calculos utilizando variaveis como em: x2. utilizando a tag sup

já em texto subscritos são utilizados com ituito de deixar os numeros embaixo de letras variaveis como em elementos químicos tipo: H2O. utilizando a tag sub
